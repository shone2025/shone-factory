# ShoneFactory Token Key

> Factory Token 管理客户端

## 快速开始

### Windows
```
双击 windows-start.bat
```

### macOS / Linux
```bash
chmod +x mac-start.sh
./mac-start.sh
```

### 或直接运行
```bash
python3 shone_client_web.py
```

## 使用方法

1. 获取 SF-Key（联系作者）
2. 粘贴 SF-Key 到输入框
3. 点击「➕ 点击添加」
4. 首次添加自动完成配置

## SF-Key 格式

```
SF-01xxxxxxxxxxxxxxxxxxxxxxxxxxxx
SF-02xxxxxxxxxxxxxxxxxxxxxxxxxxxx
SF-00xxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

每行以 `SF-` 开头，支持多行粘贴。

## 环境要求

- Python 3.8+
- Factory (droid) 已安装
